# НМБК — концепт-редизайн сайта

Концепт-редизайн официального сайта **ГБПОУ НО «Нижегородский медицинский колледж»** ([nmbc.ru](http://nmbc.ru/)).

**Live:** https://averjahn.github.io/nmbc-redesign/

## Что сделано
- **Светлая медицинская палитра** (мятно-бирюзовая) вместо пёстрых цветов оригинала.
- **Удобная навигация** по обширному порталу: липкое мега-меню, карточки по аудиториям, карта сайта.
- **Реальный контент**, собранный с оригинального сайта (раздел «Сведения об ОО», образование, документы, стипендии, поступающему, контакты и др.).
- 22 страницы, адаптивная вёрстка, версия для слабовидящих.

## Структура
- `index.html` + страницы разделов — статический сайт (генерируется).
- `assets/` — стили, скрипт, изображения (логотип, слайды, новости, баннеры с оригинала).
- `content/` — очищенные фрагменты контента, собранные с nmbc.ru.
- `build.py` — генератор: запекает общий хедер/футер/меню во все страницы.

## Сборка
```bash
python3 build.py
```

> Учебный макет (концепт). Документы по ссылкам ведут на оригинальный сайт nmbc.ru.
